# yêu cầu 
    Máy chủ CSDL MySQL chạy ở cổng 3306
    Tạo một CSDL (Schema) có tên là NhaTro
    Cài đặt [Composer] (https://getcomposer.org/)
    Cài đặt [NodeJS] (https://nodejs.org/en/)
# Với lần khởi chạy đầu tiên
    Chạy cmd ở trong thư mục dự án
    ```bash
    composer install
    ```

    ```bash
    npm install
    ```
